# 01: Remove `/logger`

- **Date**: 13 June, 2026
- **Author**: Andrei Dumitrescu

**Problem**: `@asd14/node-utils` exposed a logger while `@asd14/fancy-logger`
already owned logging. Keeping both split the same capability across two
packages.

**Solution**: Deprecate `@asd14/node-utils/logger`, remove it from source and
exports. Use `@asd14/fancy-logger` from now on.
